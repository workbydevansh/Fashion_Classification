# Fashion_Classification
This is a end-to-end Deep Learning Project classifying different fashion styles.


This project is a lightweight image classification web app built with Streamlit and TensorFlow. It allows users to upload an image and receive a predicted clothing category using a pre-trained deep learning model based on the Fashion MNIST dataset. The application focuses on simplicity and accessibility, giving users an easy way to interact with a machine learning model through a clean browser-based interface.  When an image is uploaded, the app preprocesses it by resizing it to 28x28 pixels, converting it to grayscale, normalizing pixel values, and reshaping it to match the model’s expected input format. The processed image is then passed to a saved TensorFlow model, which predicts one of ten fashion categories, including T-shirt/top, trouser, dress, coat, sneaker, bag, and ankle boot. The predicted result is displayed instantly in the interface.  The project demonstrates the full pipeline of deploying a machine learning model in a user-facing application, from model loading and image preprocessing to real-time prediction and visualization. It also includes Docker support, making the app easier to package and deploy consistently across environments.  Overall, this project is a practical example of combining deep learning with an interactive web framework to create an end-to-end image classification system. It is well suited for showcasing applied AI, computer vision fundamentals, and basic ML deployment skills.
